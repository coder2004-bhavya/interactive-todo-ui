# Interactive To-Do Website

A modern and interactive To-Do web application built with HTML, CSS, and JavaScript.  
It helps users add, manage, and track daily tasks with a clean UI and local storage support.

## Features

- Add new tasks
- Mark tasks as completed or active
- Edit existing tasks
- Delete tasks
- Filter by `All`, `Active`, and `Completed`
- Clear all completed tasks
- View remaining active task count
- Automatic data persistence with browser `localStorage`

## Project Structure

- `index.html` - App structure and layout
- `styles.css` - UI design and responsive styling
- `script.js` - To-Do logic and interactivity

## Run Locally

1. Download or clone this repository.
2. Open `index.html` in any modern browser.
3. Start managing your tasks.

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Push this project code to the repository.
3. Go to repository `Settings` -> `Pages`.
4. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main` (or your default branch), folder: `/ (root)`
5. Save and wait for deployment.
6. Your live website URL will appear in the GitHub Pages section.
